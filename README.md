# team03
바보 대회 팀3의 키보드 프로젝트입니다.

## 실행 방법

`cursed-chat.jsx`는 Vite + React + Tailwind CSS 프로젝트로 세팅되어 있고,
그 내용은 `src/App.jsx`에도 동일하게 복사되어 있습니다.

```bash
npm install   # 처음 한 번만
npm run dev
```

명령 실행 후 터미널에 나오는 주소(기본 `http://localhost:5173`)를 브라우저로 열면 됩니다.

종료는 `Ctrl+C`.

<미안, 사자가 눌렀어>
사자가 밟고 지나가 단 한 줄도 똑바로 못 치는 대환장 오타 채팅창!
당신의 인내심을 테스트하세요!


기능 1. [혈압주의] 사자의 키보드 가로채기
타이핑하는 순간, 바보 사자가 튀어나와 엉뚱한 키를 꾹 눌러버리는 대리 오타 기능.

기능 2. [우당탕탕] 사자가 굴리는 전송 버튼
전송 좀 하겠다는데 사자가 앞발로 버튼을 뻥뻥 찹니다! 화면 여기저기 제멋대로 굴러다니는 전송 버튼을 끝까지 쫓아가서 클릭해야 하는 기능.

기능 3. [낙장불입] 백스페이스 먹통 기능
오타를 지우려고 백스페이스를 눌러도 여러 번 무한 연타해야 겨우 한 글자 지워질까 말까 한 인내심 테스트 기능.
